# Haldiram Supply Chain Dashboard (Power BI)

## Project Overview
This project analyzes the supply chain performance of Haldiram using Power BI. The dashboard helps track inventory, orders, and delivery performance.

Note: Dummy dataset is used for analysis.

## Tools Used
Power BI
Excel
Data Visualization

## Dashboard Insights
- Total Orders
- Inventory levels
- Supplier performance
- Delivery status
- Product category analysis

## Files Included
Haldiram_SupplyChain_Dashboard.pbix – Power BI dashboard  
dataset.xlsx – Dummy dataset used  
dashboard.png – Dashboard preview

## Purpose
This project demonstrates data visualization and supply chain analysis skills using Power BI.
